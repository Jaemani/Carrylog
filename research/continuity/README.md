# Continuity research

This directory contains the evidence program governed by ADR-0012.

- `external-audit-brief-2026-07-12.md`: broad question and threat inventory supplied for review;
- `external-review-2026-07-12.md`: advisory findings and maintainer disposition;
- `tier1-pilot-protocol.md`: minimum executable pilot before journaling or compaction expansion.

The audit brief is intentionally broader than the Tier 1 protocol. Tier 1 is the next executable
step; the larger design is not a prerequisite for the narrow checkpoint/resume beta.
